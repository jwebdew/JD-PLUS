그라디언트 메쉬(Gradient Mash)

object - Create Gradient Mesh

1. 컬러 선택, 색상 조합

그라데이션 적용할 색상을 고른 후 컬러칩으로 정리

2. 그라디언트 메쉬

적용할 오브젝트 선택 후 그라데이션 메쉬 선택

object - Create Gradient Mesh

3. 그라디언트 메쉬 앵커포인트 지정

행과 열의 수를 지정하여 오브젝트 분할, 앵커포인트를 이용하여 색상적용

이후 메쉬 도구툴(U) 선택하고 도형 안쪽에 앵커포인트 추가 가능

4. 메쉬에 색상 적용

직접선택툴(A)를 선택 후 메쉬의 앵커포인트 클릭 후 색상 적용

컬러패널에서 사용자가 선택하거나 또는 도형툴에 설정해놓은 색상을 스포이트툴(I) 이용

5. 앵커 포인트 조정

직접선택툴(A)을 선택 후 각 앵커 포인트를 움직여 주위 이동 가능

역동적으로 만들고 생조의 농도 조정 가능

6. 완성

다채로운 그라데이션 완성
